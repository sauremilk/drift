# Language Roadmap

Python = Tier 1 (full-semantic, v0.1+)
TypeScript = Tier 1 (full-semantic, v0.5+)
JavaScript = Tier 1 (full-semantic, v0.5+, via TS parser)
Go = Tier 3 (planned)

Kotlin = Tier 4 (config-only planned)

Go work may begin once TS/JS support is validated in production.
