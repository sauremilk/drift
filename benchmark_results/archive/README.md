# Archived Benchmark Artifacts

Historical benchmark results moved here to reduce workspace noise.
These files are **not referenced by active tests, CI, or documentation**.

| Directory | Contents |
|-----------|----------|
| `single_repo/` | Per-repository benchmark runs (celery, django, fastapi, …) |
| `temporal/` | Temporal drift analysis results |
| `validation/` | Ground-truth labels and holdout validation |
| `deprecated/` | Pre-ADR snapshots, old baselines, v0.1 backups |
| `feature_evidence/` | Feature evidence files for versions v0.7–v1.1 |

**Do not** use these files for new features or analysis.
The active benchmark artifacts remain in the parent directory.
